# GOBS Golf

Web-based scoring and league management app for the Good Ole Boys
golf league at Semiahmoo Golf & Country Club.

## Canonical project documents

- [ROADMAP.md](./ROADMAP.md) — feature roadmap. Source of truth for *what* to build,
  organized by phase. Tracks status, decisions locked, open questions.
- [GOBS_Game_Rules_v1.pdf](./GOBS_Game_Rules_v1.pdf) — game rules document. Source
  of truth for *how* scoring works (formats, handicap application, blind draw, money).

## Getting started

```bash
npm run dev
```

Then open [http://localhost:3000](http://localhost:3000).
