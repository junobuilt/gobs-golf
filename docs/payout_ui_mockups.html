Edit Engine Constants (Future / Rare)
Edit Payout Rules?
Changing payout rules will only affect future rounds. Past rounds remain unchanged.

1st place cap: $25/player
Min floor: $5/player
Min gap: $3/player
Proportions: 50/25/17/8

These defaults come from your locked rule set. Edit with care.
